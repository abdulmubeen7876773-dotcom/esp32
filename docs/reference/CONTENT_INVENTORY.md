## Architecture after cleanup

```
content/
  guides/*.yaml          ─┐
  components/*.yaml      ─┼─►  py tools/build_all.py  ─►  HTML + JSON + XML + JS
  projects/*.yaml        ─┘         (tools/content_store.py)

projects/_archive/       ───►  Fallback wiring only (via tools/legacy/export_project_hardware.py)
                               Primary wiring now in content/projects/*.yaml → hardware:
```

**Build pipeline reads content exclusively through `get_content_store()`** — not from `projects.json` or generated HTML.
