DHT22 Temperature & Humidity Sensor

Measures temperature and humidity in the air. Perfect for weather stations and room monitors.

Explain Like I'm 12

Think of DHT22 like a tiny weather reporter inside a plastic box. It tells your ESP32 how warm and how damp the air is — just like checking if your room feels cozy or sticky.

Specifications

  • Temperature range: -40°C to 80°C
  • Humidity range: 0–100% RH
  • Operating voltage: 3.3 V to 5 V
  • Single data pin
  • Update rate: every 2 seconds

Pinout & Wiring

  • VCC → 3.3 V
  • GND → GND
  • DATA → GPIO4 (with 10kΩ pull-up to 3.3 V)

Library

DHT sensor library by Adafruit

Example Code

#include "DHT.h"
#define DHTPIN 4
#define DHTTYPE DHT22
DHT dht(DHTPIN, DHTTYPE);
void setup() {
  Serial.begin(115200);
  dht.begin();
}
void loop() {
  float t = dht.readTemperature();
  float h = dht.readHumidity();
  Serial.printf("Temp: %.1f°C  Humidity: %.0f%%\n", t, h);
  delay(2000);
}

Expected Output

Serial Monitor shows live temperature and humidity readings every 2 seconds.
