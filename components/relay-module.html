Motors Beginner

1-Channel Relay Module

Switches high-power devices like fans, pumps, and lights using a small ESP32 signal.

BeginnerDifficulty 12 minReading time 20-35 minBench time ESP32Compatible

00Plain-English Overview

A relay is like a remote-controlled light switch. Your ESP32 sends a tiny signal and the relay clicks ON or OFF to control something big — like a water pump or a lamp.

01Quick Facts

  • Category Motors
  • Level Beginner
  • Use with ESP32

02Technical Specifications

Arduino library: Built-in digitalWrite

  • Control voltage: 3.3 V or 5 V
  • Switch rating: up to 10 A at 250 V AC
  • Opto-isolated input
  • Active LOW on most modules

03Pinout

  • VCC → 5 V
  • GND → GND
  • IN → GPIO26

04Wiring Diagram

Connect each pin as shown in the pinout section above.

05Code Examples

Use the same wiring with Arduino IDE, PlatformIO, or ESP-IDF. Start with Arduino, then graduate when you need a larger project structure.

example.ino 
#define RELAY 26
void setup() { pinMode(RELAY, OUTPUT); digitalWrite(RELAY, HIGH); }
void loop() {
  digitalWrite(RELAY, LOW);
  delay(2000);
  digitalWrite(RELAY, HIGH);
  delay(2000);
}
