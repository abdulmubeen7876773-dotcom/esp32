ESP32 Energy Meter for Power Scan

Energy MonitoringIntermediate11 min read

Automate Current Sensor control with ESP32 by reacting to live Voltage Sensor signal readings — ideal for Current Logger.

This tutorial walks through energy meter for power scan — wiring, firmware, and a quick bench test.

This build focuses on stable Energy Monitoring behavior: sense with Voltage Sensor signal, act with Current Sensor control.

Built for current logger: extend later with I2C Bus if you need remote visibility.

Project Overview

Designed for Current Logger: ESP32 watches Voltage Sensor signal and activates Current Sensor control when conditions cross your threshold.

  • Voltage Sensor signal connects to GPIO36 (ADC input)
  • Threshold is set to 2630 in firmware (adjust after calibration)
  • Current Sensor control is driven on GPIO39 when the condition is met
  • The main loop samples every 540 ms while powered

Parts Required

Components needed for this build:

  • ESP32 Dev Board
  • Voltage Sensor
  • Current Sensor
  • Lcd Display

How It Works

01

Read

The ESP32 samples the input sensor on GPIO36, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO39 high; otherwise it stays low.

04

Act

When triggered, Current Sensor control on GPIO39 turns on until the reading returns below the threshold.

Arduino Code

Upload this sketch in the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_smart_energy_meter_with_bluetooth_.ino
// ESP32 Energy Meter for Power Scan
const int SENSOR_PIN = 36;
const int OUTPUT_PIN = 39;
const int THRESHOLD = 2837;
const int SAMPLE_DELAY_MS = 540;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: energy meter for power scan");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Voltage Sensor signal: ");
  Serial.println(reading);
  bool active = reading < THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
