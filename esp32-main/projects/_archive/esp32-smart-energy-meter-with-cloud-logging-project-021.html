ESP32 Energy Meter — Power Meter Tutorial with Cloud Sync

FeaturedEnergy MonitoringIntermediate8 min read

Full-stack Energy Monitoring build: wire Voltage Sensor signal, flash firmware, and validate Current Sensor control on GPIO39 in under 7 minutes.

This tutorial walks through energy meter — power meter tutorial with cloud sync — wiring, firmware, and a quick bench test.

This build focuses on stable Energy Monitoring behavior: sense with Voltage Sensor signal, act with Current Sensor control.

Built for load logger: extend later with GPIO Interrupt if you need remote visibility.

For Load Logger, log ten minutes of idle readings before picking THRESHOLD so false triggers from noise stay rare.

Project Overview

The firmware keeps logic local on the ESP32, making it dependable for Load Logger prototypes.

  • Voltage Sensor signal connects to GPIO36 (ADC input)
  • Threshold is set to 3517 in firmware (adjust after calibration)
  • Current Sensor control is driven on GPIO39 when the condition is met
  • The main loop samples every 1270 ms while powered

Wiring tips

  • Keep sensor signal wires short and away from relay coils to reduce ADC noise.
  • Share GND between ESP32, sensor module, and relay board — never leave grounds floating.
  • Use the 3.3 V pin for logic-level modules; power heavy loads through a separate 5 V supply and relay.
  • Double-check GPIO36 and GPIO39 in both the wiring diagram and sketch before uploading.

How It Works

01

Read

The ESP32 samples the input sensor on GPIO36, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO39 high; otherwise it stays low.

04

Act

Current Sensor control activates on GPIO39 for each threshold crossing — watch the Serial log to confirm timing.

Arduino Code

Upload this sketch in the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_smart_energy_meter_with_cloud_logg.ino
// ESP32 Energy Meter — Power Meter Tutorial with Cloud Sync
const int SENSOR_PIN = 36;
const int OUTPUT_PIN = 39;
const int THRESHOLD = 3724;
const int SAMPLE_DELAY_MS = 1270;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: energy meter — power meter tutorial with cloud sync");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Voltage Sensor signal: ");
  Serial.println(reading);
  bool active = reading < THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
