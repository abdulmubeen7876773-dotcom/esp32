Contact

We welcome questions, feedback, tutorial corrections, and topic suggestions. For the fastest response, include the project page URL and a short description of your issue.

How to reach us

  • Email: contact@esp32engine.com (general inquiries and AdSense or partnership questions)
  • GitHub Issues: Report bugs or suggest improvements
  • Repository: View source on GitHub

Response time

We aim to respond within 2–5 business days. Tutorial corrections that affect wiring safety or code accuracy are prioritized.

What to include

  • Project name or URL
  • ESP32 board model and sensor modules used
  • Steps you already tried and any error messages
