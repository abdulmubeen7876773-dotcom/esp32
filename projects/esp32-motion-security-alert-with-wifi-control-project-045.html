ESP32 Motion Security Alert for Storage Room Monitoring v2

Security Projects · Advanced · Project

Build an ESP32 based motion security alert that detects data using a PIR motion sensor and provides reliable automation for real-world use.

This tutorial explains how to build ESP32 Motion Security Alert for Storage Room Monitoring v2 from wiring to working firmware.

In this project, the ESP32 reads Pir Motion Sensor signal on GPIO33 and drives Buzzer control on GPIO25 when the threshold is crossed.

ESP32 Motion Security Alert for Storage Room Monitoring v2 is designed as a practical security projects build you can test on a breadboard and later expand with Wi-Fi or cloud features.

Project Overview

Build an ESP32 based motion security alert that detects data using a PIR motion sensor and provides reliable automation for real-world use.

  • The ESP32 connects to your circuit and reads the input sensor;
  • Sensor values are compared against a threshold in the sketch;
  • When the condition is met, the output device is activated on GPIO25;
  • The loop repeats while the board is powered.

How It Works

01

Read

The ESP32 samples the input sensor on GPIO33, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO25 high; otherwise it stays low.

04

Act

ESP32 continuously reads the PIR motion sensor. When the reading crosses the configured threshold, it activates the buzzer and updates the local output for the user.

Code

Upload the following sketch to your ESP32 board using the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_motion_security_alert_with_wifi_co.ino
// ESP32 Motion Security Alert for Storage Room Monitoring v2
const int sensorPin = 33;
const int outputPin = 25;
void setup() { Serial.begin(115200); pinMode(outputPin, OUTPUT); }
void loop() { int value = analogRead(sensorPin); Serial.println(value); digitalWrite(outputPin, value > 2000 ? HIGH : LOW); delay(1000); }
