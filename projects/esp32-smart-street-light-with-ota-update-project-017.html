ESP32 Smart Street Light with OTA update Project 017 is a practical intermediate ESP32 project for smart city use cases. Build an ESP32 based smart street light that monitors data using a LDR sensor and provides reliable automation for real-world use. Instead of being only a classroom circuit, this build is designed like a small real-world automation system: it collects a reading, checks the condition, and then responds through connected hardware. That makes it useful for students who need a complete project idea, hobbyists who want something they can assemble on a breadboard, and engineers who want a simple prototype before building a larger IoT solution.

The main idea is simple. The ESP32 acts as the controller, while the Ldr Sensor provides the input signal. ESP32 continuously reads the LDR sensor. When the reading crosses the configured threshold, it activates the street light relay and updates the local output for the user. After the value is read, the firmware compares it with a configured limit or decision rule. When the condition is matched, the ESP32 drives the Street Light Relay, sends a status update, or makes the next part of the circuit react. Because the logic runs directly on the board, the project can continue working locally even when WiFi is not available, while internet features can be added later for dashboards or alerts.

For hardware, the project uses common parts such as ESP32 Dev Board, Ldr Sensor, Street Light Relay, Motion Sensor, Jumper Wires. These components are easy to find and suitable for repeated testing. The wiring should be done carefully, especially when a relay, motor, pump, or external supply is involved. Always connect ground correctly, keep signal wires short during testing, and power high-current devices separately from the ESP32 when required. Before final installation, upload the sketch, open the Serial Monitor, and confirm that the raw values change when the sensor condition changes.

This project can be applied in Smart City, IoT learning, automation prototypes. Its strongest benefits are Low cost, WiFi ready, improves reliability. The same pattern can also be reused in many other ESP32 projects: replace the input sensor, adjust the threshold, and change the output device. That reuse makes this build a good learning base for automation, monitoring, and connected product ideas.

// ESP32 Smart Street Light with OTA update Project 017
const int sensorPin = 33;
const int outputPin = 25;
void setup() { Serial.begin(115200); pinMode(outputPin, OUTPUT); }
void loop() { int value = analogRead(sensorPin); Serial.println(value); digitalWrite(outputPin, value > 2000 ? HIGH : LOW); delay(1000); }
