ESP32 Smart Street Light with OTA update Project 009 is a practical intermediate ESP32 project for smart city use cases. Build an ESP32 based smart street light that monitors data using a LDR sensor and provides reliable automation for real-world use. Instead of being only a classroom circuit, this build is designed like a small real-world automation system: it collects a reading, checks the condition, and then responds through connected hardware. That makes it useful for students who need a complete project idea, hobbyists who want something they can assemble on a breadboard, and engineers who want a simple prototype before building a larger IoT solution.

The main idea is simple. The ESP32 acts as the controller, while the Ldr Sensor provides the input signal. ESP32 continuously reads the LDR sensor. When the reading crosses the configured threshold, it activates the street light relay and updates the local output for the user. After the value is read, the firmware compares it with a configured limit or decision rule. When the condition is matched, the ESP32 drives the Street Light Relay, sends a status update, or makes the next part of the circuit react. Because the logic runs directly on the board, the project can continue working locally even when WiFi is not available, while internet features can be added later for dashboards or alerts.

Project Overview

How It Works

01

Read

The ESP32 samples the input sensor on GPIO33, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO25 high; otherwise it stays low.

04

Act

ESP32 continuously reads the LDR sensor. When the reading crosses the configured threshold, it activates the street light relay and updates the local output for the user.

Code

Upload the following sketch to your ESP32 board using the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_smart_street_light_with_ota_update.ino
// ESP32 Smart Street Light with OTA update Project 009
const int sensorPin = 33;
const int outputPin = 25;
void setup() { Serial.begin(115200); pinMode(outputPin, OUTPUT); }
void loop() { int value = analogRead(sensorPin); Serial.println(value); digitalWrite(outputPin, value > 2000 ? HIGH : LOW); delay(1000); }
