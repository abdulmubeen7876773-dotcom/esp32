ESP32 Mini Weather Station

Build a mini weather station that reads temperature and humidity, then shows the numbers on your computer.

BeginnerAges 10+45–60 minUnder $20Parent Safe
🌤
🌤 Project Mission

Build Your Mini Weather Station

📖The Story

Your desk needs a weather reporter. Not a phone app — a real sensor you wired yourself.

In this project you connect a DHT22 to your ESP32 and watch live temperature and humidity numbers appear on your screen. That's the same building block used in classroom weather stations and greenhouse monitors.

🧒Explain Like I'm 12

The DHT22 feels the air — how warm it is and how damp it feels. Your ESP32 reads those numbers and sends them to your laptop through the USB cable, like a tiny weather text message every two seconds.

For Parents & Teachers

This project uses 3.3 V USB power only — no mains electricity. Children should have an adult nearby for USB connections and breadboard wiring. Unplug before changing wires.

🛠What You Will Build

A breadboard circuit with a DHT22 sensor and ESP32. Your code reads temperature (°C) and humidity (%) every two seconds and prints them to Serial Monitor.

🔗Wiring

Use a 3-pin DHT22 module. Match each wire before plugging in USB.

  1. 1

    Unplug USB. Place the ESP32 and DHT22 module on your breadboard.

  2. 2

    Connect VCC on the module to 3.3 V on the ESP32.

  3. 3

    Connect GND on the module to GND on the ESP32.

  4. 4

    Connect DATA on the module to GPIO4 on the ESP32.

  5. 5

    Double-check all three wires, then plug in USB.

⌨️Code

Copy into Arduino IDE. Install any libraries noted in the component guides first.

mini_weather_station.ino 
#include "DHT.h"

#define DHTPIN 4
#define DHTTYPE DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();
  Serial.println("Mini Weather Station ready!");
}

void loop() {
  float temp = dht.readTemperature();
  float humidity = dht.readHumidity();

  if (isnan(temp) || isnan(humidity)) {
    Serial.println("Read failed — check wiring");
  } else {
    Serial.print("Temperature: ");
    Serial.print(temp);
    Serial.print(" C  |  Humidity: ");
    Serial.print(humidity);
    Serial.println(" %");
  }

  delay(2000);
}

✨Expected Output

Open Serial Monitor at 115200 baud. Every 2 seconds you should see:

Temperature: 24.3 C | Humidity: 55 %

Breathe on the sensor or hold it near a window — the numbers should change.

🔧Troubleshooting

  • Serial shows Read failed Check VCC, GND, and DATA wiring. Confirm GPIO4 in code matches your wire.
  • No output in Serial Monitor Set baud rate to 115200 and select the correct COM port for your ESP32.
  • Numbers look wrong Wait 30 seconds after power-on. Keep the sensor away from your breath for a stable reading.

⚠️Common Mistakes

  • Reading faster than every 2 seconds — the DHT22 needs a pause between reads.
  • DATA wire on the wrong GPIO pin compared to your code.
  • Using a charge-only USB cable that cannot upload code.
  • Loose breadboard connections that look connected but are not.

⬆️Upgrade Ideas

  • Add an OLED display to show readings without a laptop
  • Log readings to SD card every minute
  • Send data to a web dashboard over Wi-Fi (Advanced)
  • Trigger a fan when humidity goes above 70%
