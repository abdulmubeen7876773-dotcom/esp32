ESP32 IoT Weather Station for Data Logger Automation

IoT Projects · Beginner · 5 min read · Project

Monitor Bme280 Sensor signal, set a threshold, and let ESP32 manage Cloud Dashboard control without cloud dependency.

This tutorial walks through IoT weather station for data logger automation — wiring, firmware, and a quick bench test.

The sketch samples Bme280 Sensor signal on GPIO21, compares the value, and toggles Cloud Dashboard control on GPIO22.

Built for connected hub: extend later with ADC Sampling if you need remote visibility.

Project Overview

This IoT project uses Bme280 Sensor signal on GPIO21 to decide when Cloud Dashboard control on GPIO22 should run — suited for Connected Hub.

  • Bme280 Sensor signal connects to GPIO21 (ADC input)
  • Threshold is set to 3208 in firmware (adjust after calibration)
  • Cloud Dashboard control is driven on GPIO22 when the condition is met
  • The main loop samples every 831 ms while powered

How It Works

01

Read

The ESP32 samples the input sensor on GPIO21, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO22 high; otherwise it stays low.

04

Act

GPIO GPIO22 mirrors the decision: HIGH activates Cloud Dashboard control, LOW idles the circuit.

Code

Upload the following sketch to your ESP32 board using the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_iot_weather_station_with_ota_updat.ino
// ESP32 IoT Weather Station for Data Logger Automation
const int SENSOR_PIN = 21;
const int OUTPUT_PIN = 22;
const int THRESHOLD = 3415;
const int SAMPLE_DELAY_MS = 831;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: iot weather station for data logger automation");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Bme280 Sensor signal: ");
  Serial.println(reading);
  bool active = reading > THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
