ESP32 Irrigation Controller with Soil Moisture

AgricultureBeginner7 min read

Monitor Soil Moisture Sensor signal, set a threshold, and let ESP32 manage Water Pump control without cloud dependency.

This tutorial walks through irrigation controller with soil moisture — wiring, firmware, and a quick bench test.

This build focuses on stable Agriculture behavior: sense with Soil Moisture Sensor signal, act with Water Pump control.

Built for water tank: extend later with OLED Display if you need remote visibility.

Project Overview

This Agriculture project uses Soil Moisture Sensor signal on GPIO34 to decide when Water Pump control on GPIO26 should run — suited for Water Tank.

  • Soil Moisture Sensor signal connects to GPIO34 (ADC input)
  • Threshold is set to 1674 in firmware (adjust after calibration)
  • Water Pump control is driven on GPIO26 when the condition is met
  • The main loop samples every 1755 ms while powered

How It Works

01

Read

The ESP32 samples the input sensor on GPIO34, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO26 high; otherwise it stays low.

04

Act

GPIO GPIO26 mirrors the decision: HIGH activates Water Pump control, LOW idles the circuit.

Arduino Code

Upload this sketch in the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_smart_irrigation_controller_with_b.ino
// ESP32 Irrigation Controller with Soil Moisture
const int SENSOR_PIN = 34;
const int OUTPUT_PIN = 26;
const int THRESHOLD = 1881;
const int SAMPLE_DELAY_MS = 1755;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: irrigation controller with soil moisture");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Soil Moisture Sensor signal: ");
  Serial.println(reading);
  bool active = reading < THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
