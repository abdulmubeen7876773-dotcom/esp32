ESP32 Rgb Led Pattern Controller — Pixel Node Tutorial

LED Projects · Intermediate · 8 min read · Project

A compact ESP32 circuit for Color Mix that links Ws2812B Led Strip signal input to Push Button control action.

This tutorial walks through rgb led pattern controller — pixel node tutorial — wiring, firmware, and a quick bench test.

This build focuses on stable LED behavior: sense with Ws2812B Led Strip signal, act with Push Button control.

Built for color mix: extend later with WiFi Link if you need remote visibility.

Project Overview

The firmware keeps logic local on the ESP32, making it dependable for Color Mix prototypes.

  • Ws2812B Led Strip signal connects to GPIO13 (ADC input)
  • Threshold is set to 1741 in firmware (adjust after calibration)
  • Push Button control is driven on GPIO14 when the condition is met
  • The main loop samples every 1510 ms while powered

How It Works

01

Read

The ESP32 samples the input sensor on GPIO13, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO14 high; otherwise it stays low.

04

Act

Push Button control activates on GPIO14 for each threshold crossing — watch the Serial log to confirm timing.

Code

Upload the following sketch to your ESP32 board using the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_rgb_led_pattern_controller_for_low.ino
// ESP32 Rgb Led Pattern Controller — Pixel Node Tutorial
const int SENSOR_PIN = 13;
const int OUTPUT_PIN = 14;
const int THRESHOLD = 1948;
const int SAMPLE_DELAY_MS = 1510;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: rgb led pattern controller — pixel node tutorial");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Ws2812B Led Strip signal: ");
  Serial.println(reading);
  bool active = reading < THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
