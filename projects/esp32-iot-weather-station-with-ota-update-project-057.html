ESP32 IoT Weather Station and Cloud Dashboard for Signal Bridge

IoT Projects · Beginner · 6 min read · Project

Automate Cloud Dashboard control with ESP32 by reacting to live Bme280 Sensor signal readings — ideal for Subscribe Hub.

This tutorial walks through IoT weather station and cloud dashboard for signal bridge — wiring, firmware, and a quick bench test.

The sketch samples Bme280 Sensor signal on GPIO21, compares the value, and toggles Cloud Dashboard control on GPIO22.

Built for subscribe hub: extend later with Flash Config if you need remote visibility.

Project Overview

Designed for Subscribe Hub: ESP32 watches Bme280 Sensor signal and activates Cloud Dashboard control when conditions cross your threshold.

  • Bme280 Sensor signal connects to GPIO21 (ADC input)
  • Threshold is set to 1364 in firmware (adjust after calibration)
  • Cloud Dashboard control is driven on GPIO22 when the condition is met
  • The main loop samples every 1316 ms while powered

Parts Required

Here's a list of parts needed to build this project:

How It Works

01

Read

The ESP32 samples the input sensor on GPIO21, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO22 high; otherwise it stays low.

04

Act

When triggered, Cloud Dashboard control on GPIO22 turns on until the reading returns below the threshold.

Code

Upload the following sketch to your ESP32 board using the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_iot_weather_station_with_ota_updat.ino
// ESP32 IoT Weather Station and Cloud Dashboard for Signal Bridge
const int SENSOR_PIN = 21;
const int OUTPUT_PIN = 22;
const int THRESHOLD = 1571;
const int SAMPLE_DELAY_MS = 1316;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: iot weather station and cloud dashboard for signal bridge");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Bme280 Sensor signal: ");
  Serial.println(reading);
  bool active = reading > THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
