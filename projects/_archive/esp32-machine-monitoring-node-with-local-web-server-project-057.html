ESP32 Machine Monitoring Node and Relay Output for Process Monitor

Industrial AutomationBeginner6 min read

Automate Relay Output control with ESP32 by reacting to live Current Sensor signal readings — ideal for Line Logger.

This tutorial walks through machine monitoring node and relay output for process monitor — wiring, firmware, and a quick bench test.

The sketch samples Current Sensor signal on GPIO35, compares the value, and toggles Relay Output control on GPIO19.

Built for line logger: extend later with Flash Config if you need remote visibility.

Project Overview

Designed for Line Logger: ESP32 watches Current Sensor signal and activates Relay Output control when conditions cross your threshold.

  • Current Sensor signal connects to GPIO35 (ADC input)
  • Threshold is set to 1364 in firmware (adjust after calibration)
  • Relay Output control is driven on GPIO19 when the condition is met
  • The main loop samples every 1316 ms while powered

Parts Required

Components needed for this build:

  • ESP32 Dev Board
  • Current Sensor
  • Relay Output
  • Alarm Led

How It Works

01

Read

The ESP32 samples the input sensor on GPIO35, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO19 high; otherwise it stays low.

04

Act

When triggered, Relay Output control on GPIO19 turns on until the reading returns below the threshold.

Arduino Code

Upload this sketch in the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_machine_monitoring_node_with_local.ino
// ESP32 Machine Monitoring Node and Relay Output for Process Monitor
const int SENSOR_PIN = 35;
const int OUTPUT_PIN = 19;
const int THRESHOLD = 1571;
const int SAMPLE_DELAY_MS = 1316;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: machine monitoring node and relay output for process monitor");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Current Sensor signal: ");
  Serial.println(reading);
  bool active = reading > THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
