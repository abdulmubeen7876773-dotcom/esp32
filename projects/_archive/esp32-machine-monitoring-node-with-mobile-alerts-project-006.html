ESP32 Machine Monitoring Node with Relay Output Control for Line Sensor

FeaturedIndustrial AutomationBeginner8 min read

Flagship ESP32 tutorial for Process Unit: calibrated sensing, clean GPIO routing, and Serial-verified Relay Output control control.

This tutorial walks through machine monitoring node with relay output control for line sensor — wiring, firmware, and a quick bench test.

You'll connect Current Sensor signal to GPIO35, configure a threshold, and drive Relay Output control on GPIO19.

Built for process unit: extend later with Local Server if you need remote visibility.

For Process Unit, log ten minutes of idle readings before picking THRESHOLD so false triggers from noise stay rare.

Project Overview

The firmware keeps logic local on the ESP32, making it dependable for Process Unit prototypes.

  • Current Sensor signal connects to GPIO35 (ADC input)
  • Threshold is set to 1992 in firmware (adjust after calibration)
  • Relay Output control is driven on GPIO19 when the condition is met
  • The main loop samples every 494 ms while powered

Wiring tips

  • Use the 3.3 V pin for logic-level modules; power heavy loads through a separate 5 V supply and relay.
  • Double-check GPIO35 and GPIO19 in both the wiring diagram and sketch before uploading.
  • Add a 100 nF capacitor near the sensor VCC pin if readings jump when Wi-Fi transmits.
  • Keep sensor signal wires short and away from relay coils to reduce ADC noise.

How It Works

01

Read

The ESP32 samples the input sensor on GPIO35, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO19 high; otherwise it stays low.

04

Act

Relay Output control activates on GPIO19 for each threshold crossing — watch the Serial log to confirm timing.

Arduino Code

Upload this sketch in the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_machine_monitoring_node_with_mobil.ino
// ESP32 Machine Monitoring Node with Relay Output Control for Line Sensor
const int SENSOR_PIN = 35;
const int OUTPUT_PIN = 19;
const int THRESHOLD = 2199;
const int SAMPLE_DELAY_MS = 494;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: machine monitoring node with relay output control for line sensor");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Current Sensor signal: ");
  Serial.println(reading);
  bool active = reading > THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
