ESP32 Pulse Oximeter Logger — Step-by-Step Guide with Timer Trigger

HealthcareAdvanced15 min read

Step-by-step guide: connect Max30102 Sensor signal, upload the sketch, and test Oled Display control in Health Node.

This tutorial is for educational purposes only and is not medical advice. Do not use it for diagnosis, treatment, or patient monitoring without proper validation.

This tutorial walks through pulse oximeter logger — step-by-step guide with timer trigger — wiring, firmware, and a quick bench test.

You'll connect Max30102 Sensor signal to GPIO21, configure a threshold, and drive Oled Display control on GPIO22.

Built for health node: extend later with EEPROM Store if you need remote visibility.

Project Overview

This Healthcare project uses Max30102 Sensor signal on GPIO21 to decide when Oled Display control on GPIO22 should run — suited for Health Node.

  • Max30102 Sensor signal connects to GPIO21 (ADC input)
  • Threshold is set to 3971 in firmware (adjust after calibration)
  • Oled Display control is driven on GPIO22 when the condition is met
  • The main loop samples every 1219 ms while powered

How It Works

01

Read

The ESP32 samples the input sensor on GPIO21, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO22 high; otherwise it stays low.

04

Act

GPIO GPIO22 mirrors the decision: HIGH activates Oled Display control, LOW idles the circuit.

Arduino Code

Upload this sketch in the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_pulse_oximeter_logger_with_mobile_.ino
// ESP32 Pulse Oximeter Logger — Step-by-Step Guide with Timer Trigger
const int SENSOR_PIN = 21;
const int OUTPUT_PIN = 22;
const int THRESHOLD = 1378;
const int SAMPLE_DELAY_MS = 1219;

void setup() {
  Serial.begin(115200);
  pinMode(OUTPUT_PIN, OUTPUT);
  pinMode(SENSOR_PIN, INPUT);
  Serial.println("Starting: pulse oximeter logger — step-by-step guide with timer trigger");
  Serial.print("Threshold: ");
  Serial.println(THRESHOLD);
}

void loop() {
  int reading = analogRead(SENSOR_PIN);
  Serial.print("Max30102 Sensor signal: ");
  Serial.println(reading);
  bool active = reading > THRESHOLD;
  digitalWrite(OUTPUT_PIN, active ? HIGH : LOW);
  delay(SAMPLE_DELAY_MS);
}
