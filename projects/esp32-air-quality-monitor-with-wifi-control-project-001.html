Build a Smart ESP32 Air Quality Monitor That Runs for Weeks on Battery Power v2

Environmental · Beginner · 7 min read · Project

Build an ESP32 based air quality monitor that detects data using a MQ135 gas sensor and provides reliable automation for real-world use.

This tutorial explains how to build Build a Smart ESP32 Air Quality Monitor That Runs for Weeks on Battery Power v2 from wiring to working firmware.

In this project, the ESP32 reads Mq135 Gas Sensor signal on GPIO34 and drives Fan Relay control on GPIO26 when the threshold is crossed.

Build a Smart ESP32 Air Quality Monitor That Runs for Weeks on Battery Power v2 is designed as a practical Environmental build you can test on a breadboard and later expand with Wi-Fi or cloud features.

Project Overview

Build an ESP32 based air quality monitor that detects data using a MQ135 gas sensor and provides reliable automation for real-world use.

  • The ESP32 connects to your circuit and reads the input sensor;
  • Sensor values are compared against a threshold in the sketch;
  • When the condition is met, the output device is activated on GPIO26;
  • The loop repeats while the board is powered.

How It Works

01

Read

The ESP32 samples the input sensor on GPIO34, producing a raw reading.

02

Compare

That reading is checked against a threshold configured in the sketch.

03

Decide

When the condition is met, the ESP32 sets GPIO26 high; otherwise it stays low.

04

Act

ESP32 continuously reads the MQ135 gas sensor. When the reading crosses the configured threshold, it activates the fan relay and updates the local output for the user.

Code

Upload the following sketch to your ESP32 board using the Arduino IDE. Adjust pins and threshold for your hardware.

esp32_air_quality_monitor_with_wifi_cont.ino
// Build a Smart ESP32 Air Quality Monitor That Runs for Weeks on Battery Power v2
const int sensorPin = 34;
const int outputPin = 26;
void setup() { Serial.begin(115200); pinMode(outputPin, OUTPUT); }
void loop() { int value = analogRead(sensorPin); Serial.println(value); digitalWrite(outputPin, value > 2000 ? HIGH : LOW); delay(1000); }
